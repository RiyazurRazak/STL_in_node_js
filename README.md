# Node Addons (DSA)

### Implementing STL using node addons.

### Usefull repo for beginners to understand node addons.

```bash
yarn run build
yarn run heap #for heap
yarn run stack #for stack
yarn run queue #for queue
yarn run snailsort #for snail sort
```

# Data Structures

- Stack
- Queue
- Heap (priority_queue)

# Algorithms
